# Ring Security

This Homey App enables you to add the Ring Keypad (V1) to your Homey to be used with Heimdall.

## Supported Devices

* Ring Keypad (V1)

![image][ringkeypad]

* Ring Contact Sensor (V1)

![image][ringcontact]

* Ring Motion Detector (V1)

![image][ringmotion]
